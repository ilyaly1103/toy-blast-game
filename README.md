# toy-blast-game

Проста браузерна гра "Три в ряд", реалізована на чистому JavaScript (Vanilla JS).
Проєкт використовує алгоритм Flood Fill (DFS) для пошуку та видалення груп одноколірних елементів.

## Технології
- HTML5
- CSS3
- JavaScript (ES6+)

## Як запустити проєкт (Installation & Run)

Цей проєкт не потребує встановлення додаткових залежностей або Node.js.

1. Завантажте репозиторій або склонуйте його:
   git clone (https://github.com/ilyaly1103/toy-blast-game.git)
